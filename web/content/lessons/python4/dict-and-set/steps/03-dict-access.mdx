# 3、访问 dict（字典）

我们用 dict 来存了通讯录，怎么按名字快速查到电话呢？直接用**中括号 + key**：

```python
name = {'一点水': '131456780001', '两点水': '131456780002', '三点水': '131456780003'}

print(name['两点水'])
```

输出的结果：

```
131456780002
```

可以看到，如果你知道某个人的名字（key），就能很快查到他对应的电话号码（value）。

> ⚠️ **注意**：如果字典中没有这个键，直接 `dict[key]` 会**报错**（`KeyError`）。如果不确定 key 是否存在，更安全的写法是 `dict.get(key)`——找不到时返回 `None`，不报错。

```python
name.get('六点水')   # None，不会报错
name['六点水']       # KeyError！
```
