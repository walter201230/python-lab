# 二、初始化函数 `__init__`

上一节我们的 `Person` 没有名字、没有年龄——所有 `Person` 都长得一模一样。

实际中我们希望**每个对象都带自己的属性**——比如「张三 18 岁」、「李四 25 岁」。这时候要用**初始化函数 `__init__`**。

## 1、什么是 `__init__`

`__init__` 在你**创建对象时自动被调用**——专门用来初始化新对象的属性。

写法：

```python
class 类名:
    def __init__(self, 参数1, 参数2, ...):
        self.属性1 = 参数1
        self.属性2 = 参数2
```

注意：

- `__init__` 前后**各两个下划线**——不是一个
- 第一个参数永远是 `self`
- 在 `self` 上设置的属性就是**这个对象专属**的（不同对象互不影响）

## 2、完整例子

```python
class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def greet(self):
        print(f'我叫 {self.name}，今年 {self.age} 岁')

# 实例化时传入参数
p1 = Person('张三', 18)
p2 = Person('李四', 25)

p1.greet()
p2.greet()
```

输出：

```
我叫 张三，今年 18 岁
我叫 李四，今年 25 岁
```

每个对象都有自己的 `name` 和 `age`——互不干扰。这就是 OOP 的「**封装**」效果。
