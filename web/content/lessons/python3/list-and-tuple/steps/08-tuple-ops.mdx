# 6、tuple（元组）运算符

与字符串和列表一样，元组之间可以使用 `+` 号和 `*` 号进行运算。这就意味着他们可以**组合**和**复制**，运算后会生成一个新的元组。

| Python 表达式 | 结果 | 描述 |
|-----------|-----|-----|
| `len((1, 2, 3))` | `3` | 计算元素个数 |
| `(1, 2, 3) + (4, 5, 6)` | `(1, 2, 3, 4, 5, 6)` | 连接 |
| `('Hi!',) * 4` | `('Hi!', 'Hi!', 'Hi!', 'Hi!')` | 复制 |
| `3 in (1, 2, 3)` | `True` | 元素是否存在 |
| `for x in (1, 2, 3): print(x)` | `1 2 3` | 迭代 |

# 7、元组内置函数

| 函数 | 描述 |
|----|----|
| `len(tuple)` | 计算元组元素个数 |
| `max(tuple)` | 返回元组中元素最大值 |
| `min(tuple)` | 返回元组中元素最小值 |
| `tuple(seq)` | 把可迭代对象（如 list）转换为元组 |

举个例子综合用一下：

```python
name1 = ('一点水', '两点水', '三点水', '四点水', '五点水')
list1 = [1, 2, 3, 4, 5]

# 计算元素个数
print(len(name1))
# 元素是否存在 (name1 这个元组中是否含有一点水这个元素)
print('一点水' in name1)
# 将列表转换为元组
print(tuple(list1))
```

输出：

```
5
True
(1, 2, 3, 4, 5)
```
