# 二、tuple（元组）

## 1、什么是元组（tuple）

上一节刚说了一个有序列表 List，现在说另一种有序列表叫元组：tuple。

tuple 和 List 非常类似，但是 tuple **一旦初始化就不能修改**。也就是说元组（tuple）是不可变的——它没有 `append()`、`insert()` 这样的方法，可以通过索引读，但不能赋值。

那么为什么要有 tuple 呢？

那是因为 tuple 是不可变的，所以**代码更安全**。所以建议能用 tuple 代替 list 就尽量用 tuple。

## 2、怎样创建元组（tuple）

元组创建很简单，只需要在括号中添加元素，并使用逗号隔开即可。

```python
tuple1 = ('两点水', 'twowter', 'liangdianshui', 123, 456)
tuple2 = '两点水', 'twowter', 'liangdianshui', 123, 456  # 括号也可以省
```

创建空元组：

```python
tuple3 = ()
```

元组中只包含一个元素时，**需要在元素后面添加逗号**：

```python
tuple4 = (123,)
```

如果不加逗号，创建出来的就不是元组（tuple），而是指 `123` 这个整数了。

这是因为括号 `()` 既可以表示元组，又可以表示数学公式中的小括号——产生了歧义。所以 Python 规定，单元素的时候不加逗号就按小括号算（表达式），加了逗号才是元组。

```python
tuple4 = (123,)
tuple5 = (123)
print(tuple4)
print(tuple5)
```

输出：

```
(123,)
123
```

看到区别了吗？
