# 4、break 和 continue：控制循环流程

光有循环还不够，有时候需要在循环中途**跳出**或者**跳过**：

| 语句 | 作用 |
|---|---|
| `break` | 立刻跳出**整个**循环 |
| `continue` | 跳过本次循环剩下的代码，进入下一次 |

## break：满足条件就退出

例子——累加 1+2+3+...，**一旦总和超过 1000 就停下来**：

```python
total = 0
for n in range(1, 1000):
    total = total + n
    if total > 1000:
        break

print(total)
```

输出：

```
1035
```

`break` 让循环立刻结束，不再继续往后跑。

## continue：跳过本次

例子——统计 1 到 10 中**奇数**的和（偶数跳过）：

```python
total = 0
for n in range(1, 11):
    if n % 2 == 0:
        continue   # 偶数：跳过下面的累加，直接进入下一轮
    total = total + n

print(total)
```

输出：

```
25
```

> 1+3+5+7+9 = 25 ✓

## 嵌套循环

循环可以嵌套——一个 for 里再写一个 for / while。`break` / `continue` 只对**最近的那一层**循环生效。

```python
for i in range(3):
    for j in range(3):
        if j == 2:
            break    # 只跳出 j 那层
        print(i, j)
```

输出：

```
0 0
0 1
1 0
1 1
2 0
2 1
```
