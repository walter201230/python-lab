# 2、if 多分支：elif

有些时候判断不止两个分支，比如成绩既要分及格不及格，还要细分优秀、良好。

这时候用 `elif`：

```python
if 判断条件1:
    执行语句1……
elif 判断条件2:
    执行语句2……
elif 判断条件3:
    执行语句3……
else:
    执行语句4……
```

例子：

```python
results = 89

if results > 90:
    print('优秀')
elif results > 80:
    print('良好')
elif results > 60:
    print('及格')
else:
    print('不及格')
```

输出：

```
良好
```

# 3、多个条件同时判断：and / or

要求 java 和 python 两科都 > 80 才算优秀，怎么写？

用 **`and`（与）** 和 **`or`（或）**：

- `or` 表示两个条件**有一个**成立就 True
- `and` 表示**两个都**成立才 True

```python
java = 86
python = 68

if java > 80 and python > 80:
    print('优秀')
else:
    print('不优秀')

if (java >= 80 and java < 90) or (python >= 80 and python < 90):
    print('良好')
```

输出：

```
不优秀
良好
```

> 💡 复杂条件用括号 `()` 把要先算的部分括起来，可读性最重要。
